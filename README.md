# MikoMestroMobile
Mobile Automation

## Project Structure

```
MikoMestroMobile/
├── README.md
├── Users/
│   ├── Aslam/              # Test files for Aslam
│   └── Vishakha/           # Test files for Vishakha
├── tests/
│   ├── smoke/              # Smoke tests
│   ├── negative/           # Negative tests
│   └── functional/         # Functional tests
├── pageObjects/            # Page Object Models
├── utils/                  # Utility functions
├── config/                 # Configuration files
├── reports/                # Test execution reports
└── package.json            # Project dependencies
```

## Users Directory
This directory contains test files organized by user:
- **Users/Aslam**: Contains test cases created by Aslam
- **Users/Vishakha**: Contains test cases created by Vishakha

## Getting Started
1. Install dependencies: `npm install`
2. Configure your test settings in the `config/` directory
3. Add your test files in the appropriate directories
4. Run tests and view reports in the `reports/` directory
